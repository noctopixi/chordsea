About Chordsea

what is it?

Chordsea is a minimalist chord progression generator. It is written using Python and Flask.

This is the web version. For a minimalist experience, you can also run chordsea and get ASCII tabs right in your terminal.

why is it?

When practicing guitar chord changes, I felt like I often went for the "safe", obvious pairs that I was comfortable with, and realized that I'd need a randomized practice that challenged and pushed me out of my comfort zone.

I had recently learned about the random Python module and decided to implement a guitar chord randomizer that I could run in my terminal.

I decided to create the web version to learn Flask and to make chordsea accessible to anyone who wants an easy solution to generate practice sessions for chord changes.
